# Life Hack or Urban Myth?

## Purpose
This is an Android quiz app built with Kotlin. The app presents users with statements and challenges them to decide whether each one is a real life hack or just an urban myth.

## How to Use
1. Open the app and read the welcome screen
2. Press **Start Quiz** to begin
3. For each statement, press **Hack (True)** or **Myth (False)**
4. After answering you'll get feedback on whether you were correct
5. Press **Next** to move to the next question
6. At the end you'll see your score and personalised feedback
7. Press **Review Answers** to see all the correct answers

## Screens
- **Welcome Screen** - introduces the app and has a Start button
- **Question Screen** - shows each statement with Hack/Myth buttons and feedback
- **Score Screen** - shows your final score, feedback, and a review of all answers

## GitHub Actions
This project uses GitHub Actions to automatically build the app whenever code is pushed to the main branch. The workflow file is located at `.github/workflows/build.yml`. This ensures the code compiles correctly and not just on my local machine.
